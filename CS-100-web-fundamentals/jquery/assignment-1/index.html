.addClass, .removeClass

.addClasss adds the specified class(es) to each element in the set of matched elements.
.removeClass removes the specified class(es).

.before, .after

.before insert content, specified by the parameter, before each element in the set of matched elements.
.after inserts content after each element.

.prepend, .append

.prepend inserts content, specified by the parameter, before each element in the set of matched elements.
.append inserts content after each element in the set of matched elements.

.attr, .val

.attr gets the value of an attribute for the first element in the set of matched elements or set one or more attributes for every matched element.
.val gets the current value of the first element in the set of matched elements or set the value of every matched element.

.text, .html

.text get the combined text contents of each element in the set of matched elements, including their descendants, or set the text contents of the matched elements.
.html gets the HTML contents of the first element in the set of matched elements or set the HTML contents of every matched element.

.toggle

.toggle displays or hides the matched elements. Also, bind two or more handlers to the matched elements, to be executed on alternate clicks.

.hide, .show

.hide hides the matched elements.
.show displays the matched elements.

.slideUp, .slideDown

.slideUp hides the matched elements with a sliding motion.
.slideDown displays the matched elements with a sliding motion.

.slideToggle

.slideToggle displays or hides the matched elements with a sliding motion.

There have been 0 toggled squares.

.fadeOut, .fadeIn

.fadeOut hides the matched elements by fading them to transparent.
.fadeIn displays the matched elements by fading them to opaque.

.focus

.focus binds an event handler to the “focus” JavaScript event, or trigger that event on an element.

.click

.click binds an event handler to the “click” JavaScript event, or trigger that event on an element.
